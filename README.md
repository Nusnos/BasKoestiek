# Akoestiek Calculator App

Een simpele React/Vite starter-app voor het berekenen van akoestische wandpanelen in restaurants.

## Installeren

Open deze map in VS Code of je terminal en voer uit:

```bash
npm install
npm run dev
```

Daarna open je de lokale URL die Vite toont, meestal:

```bash
http://localhost:5173
```

## Codex gebruiken

Open de map in VS Code en start Codex. Geef bijvoorbeeld deze opdracht:

> Maak van deze calculator een nette mobiele PWA. Voeg een opslaan-functie toe voor projecten, een export naar PDF en een extra tabblad waarin ik meerdere paneeltypes kan vergelijken.

## Wat zit erin?

- Ruimteberekening: lengte × breedte × hoogte
- Vuistregels: Comfort, Premium, Heavy
- Paneelberekening: paneeloppervlak × NRC
- Sabine-formule als technische basis
- Nederlandse klantuitleg
